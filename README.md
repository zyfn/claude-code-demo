# CCC — AI Coding Assistant

A collection of terminal-based AI coding assistants, each exploring different aspects of AI-assisted development.

## Sub-projects

| # | Name | Description |
|---|------|-------------|
| 01 | [Agent](projects/01-agent/) | ReAct-style think-act cycle with tool use (Anthropic + OpenAI backends) |
| 02 | [Complex Agent](projects/02-complex-agent/) | *(coming soon)* |

## Quick Start

Each sub-project is independent. Enter its directory and follow the README there.

```bash
cd projects/01-agent
pip install -e .
ccc
```
